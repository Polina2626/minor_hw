# Критерии
- Текстовые и графические гиперссылки, открывающиеся в текущем окне -- **навигация (меню)**

- Закладки - **Главная (наверх) и Каталог (содержание, наверх)**

- Гиперссылки на один или несколько внешних сайтов или страниц внешних сайтов, открывающихся в новом окне браузера -- **ссылки на соцсети под логотипом**

- Гиперссылки для перехода на начальную страницу Вашей среды, открывающуюся в текущем окне б`раузера -- **Главная и логотип**

- Навигация, показывающая посетителю его текущее положение, выделением фрагмента навигации -- **меню в шапке**

- Информационные страницы (4-6), которые должны содержать:
-- Навигацию (раздел 1, раздел 2, к началу ) внутри страницы с использование закладок -- **содержание в Каталоге**
-- Фрагменты, содержащие списки
   --- нумерованные - **Адреса**
   --- не нумерованные - **Адреса, Каталог (содержание)**
  ---  вложенные - **Адреса, Каталог (содержание)**
  
- Изображения объектов или субъектов с соответствующими пояснениями и комментариями -- **Главная**

- Изображение большого размера (600 пикселей по горизонтали и 400 по вертикали), открывающиеся в новом окне или закладке в центре как по вертикали, так и по горизонтали с использованием миниатюрной графической гиперссылки (размер по горизонтали до 100 пикселей) -- **Каталог**

- Таблицу, содержащую: изображение на заднем фоне всей таблицы, изображение в одной из ячеек, изображение, гиперссылки -- **Каталог (фон таблицы, изображение + гиперссылка в столбце "Состояние"**
